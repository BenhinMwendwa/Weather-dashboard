# 🌤️ Weather Intelligence Dashboard

A modern, mobile-first weather application that delivers real-time conditions, 7-day forecasts, dynamic visual insights, and intelligent recommendations—all wrapped in a premium glass-morphism interface.

## ✨ Key Features

- **📍 Smart Location Search**: Search any city worldwide with instant geocoding.
- **🌦️ Current Weather**: Real-time temperature, "Feels Like" index, humidity, and wind speed.
- **🧠 Smart AI-Like Insights**: Dynamic text summaries that interpret the forecast (e.g., "Rain expected on Wednesday...").
- **📅 7-Day Forecast**: A clear, at-a-glance view of upcoming weather with high/low temperatures.
- **📈 Temperature Trends**: Interactive line chart visualizing weekly temperature fluctuations (powered by Chart.js).
- **💡 Proactive Recommendations**: Actionable advice based on weather conditions (e.g., "Carry an umbrella," "Stay hydrated").
- **🎨 Dynamic Visuals**: The background gradient shifts beautifully based on the current weather condition (sunny, rainy, cloudy, etc.).
- **📱 Premium UI/UX**: Light, airy glass-morphism design with smooth animations and a mobile-first responsive layout.

---

## 🛠️ Tech Stack

- **Backend**: Python 3, Flask
- **Weather API**: Open-Meteo (Free & Open, No API Key Required)
- **Geocoding**: Open-Meteo Geocoding API
- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Visualization**: Chart.js (v4)
- **Icons**: Font Awesome (v6)
